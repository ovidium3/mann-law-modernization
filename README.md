# Mann Law Group Modernization

A modernization project for a legal-services website focused on improving user experience, multilingual accessibility, search visibility, and AI-assisted client intake workflows.

## Overview

This project redesigns and modernizes a legal-services web platform using modern frontend technologies and cloud deployment practices.

Key objectives include:

* Improve client engagement and consultation conversion
* Enhance search visibility through SEO and structured data
* Support multilingual content delivery
* Establish AI-assisted client intake workflows
* Create a scalable cloud deployment architecture

## Features

* Responsive Next.js application built with TypeScript
* Multilingual routing architecture (`/en`, `/es`, `/uk`, `/ro`)
* AI-assisted consultation intake interface
* Practice area and attorney profile pages
* FAQ search and structured content organization
* SEO optimization with metadata and schema support
* Sitemap and robots.txt generation
* Analytics integration scaffolding
* Cloudflare Pages deployment readiness
