# Mann Law Group Modernization

A modernization project for a legal services website focused on improving user experience, search visibility, multilingual accessibility, and client intake workflows.

## Features

- Modern responsive interface built with Next.js and TypeScript
- AI-powered consultation intake assistant
- Multilingual content support
- Search engine optimization (SEO) enhancements
- Automated deployment through Cloudflare Pages
- Analytics and performance monitoring
- Secure contact and consultation request workflows

## Technology Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Cloudflare Pages
- OpenAI API
- Google Analytics

## Goals

- Improve client engagement
- Streamline consultation intake
- Increase search visibility
- Provide scalable cloud-based deployment infrastructure
